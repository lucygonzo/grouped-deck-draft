# Fonts for Grouped deck

Place your font files here so the deck can use them.

**Structure:**
- `GroupedFont.otf` — Grouped heading font (in this folder)
- `tt-commons-cufonfonts/` — folder containing TT Commons .otf files:
  - `TT Commons Regular.otf`
  - `TT Commons Medium.otf`
  - `TT Commons DemiBold.otf`
  - `TT Commons Bold.otf`
  - (other weights as needed)
